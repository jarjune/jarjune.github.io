模板生成器

通过jar包的方式直接运行

  1. 在当前文件目录下，编辑generate.properties配置文件，设置好数据源、数据库等等配置
  2. 打开cmd，运行java -jar xxx.jar,默认根据时间戳来生成文件夹，所有生成的文件都放在该文件夹内

通过maven依赖运行

依赖

在resources下创建generate.properties文件，并配置好

  • generate.package 生成的文件会在这个包下
  • generate.driver jdbc驱动
  • generate.datasource 数据源
  • generate.user 数据库用户名
  • generate.pass 数据库密码

自带命令

add [index] 把index索引的表增加到生成队列
del [index] 把index索引的表从生成队列中移除
ls 查看哪些要表生成

example:
add 1
add 2

生成的目录

  • jar包运行，会生成在当前位置的时间戳文件夹下
  • 项目内部运行，会生成在generate.package下

项目结构

|— impl
|—- 表名ServiceImpl.java
|– dao
|— 表名Dao.java
|– dto
|— 表名Dto.java
|– entity
|— 表名Entity.java
